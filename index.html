Project description

OrbitTrack: Real-Time Tracking of Buses

OrbitTrack is an Android application that gives real time location updates of buses operating in the city of Tempe. Riders on the bus use the app to send their location every few seconds to the server. Users waiting for the bus at designated stops use this information to locate their transport.

Rock-Paper-Scissors for Android

The game has both single player and two player versions of the popular game. The two-player version uses bluetooth which enables users with the app to compete against each other. The application uses speech to get user input for which the google speech recognition API is utilized.

Data Mining: Remission Prediction of Depression Treatment

Classification problem to predict the treatment outcome of depressive patients. The challenge is to classify records as remission or non-remission from a large data set consisting of 600 objects and more than 2500 attributes. The classification algorithm was implemented in Matlab. Our results achieved the highest prediction accuracy among 40 competing ones.

Compiler Design

Developed a fully functional compiler for a C-like language and includes the results of Lexical, Syntactical and Semantic analysis of the source code. Generates the intermediate code of the source code. The interface also includes a virtual machine where the generated intermediate code can be input and the results of compilation can be obtained and verified. Used Java to design the compiler.
